# Drupal 8 + Commerce

An eCommerce website based on Drupal 8 and Commerce.

## Setup

Clone repository.

```shell
git clone git@bitbucket.org:dasigr/commerce.git
cd commerce
ddev config --auto
ddev composer install
ddev launch
```
